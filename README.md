Ansible role which installs and configures Prestashop

**Latest Prestashop version**

Prestashop 1.6.1.1

#### Dependencies

* MySql
* Nginx

#### Usage

```yaml
prestashop:
  nginx_site:
    server_name: prestashop.aplazame.com

  mysql_db:
    name: prestashop
    user: prestashop
    pass: "pass"
```

#### Default variables

```yaml
prestashop_version: 1.6.1.1
nginx_dir: /etc/nginx
```
